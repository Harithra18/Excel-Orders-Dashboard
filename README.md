# Product Orders Dashboard - Excel (2026)

## Overview
An interactive sales dashboard built in Microsoft Excel to analyze 
product orders across categories, cities, and payment methods.

## Key Metrics
- Total Revenue: ₹5,58,065
- Total Products Sold: 11,899
- Categories: 5 | Products: 20
- Cities Covered: 8 (UK)
  
## Features
- KPI Summary Cards (Revenue, Units Sold, Categories)
- Bar Chart: Orders by Category
- Column Chart: Revenue by Payment Method (Card/Cash/Online)
- 3D Pie Chart: Category-wise Revenue Share
- Line Chart: City-wise Total Price Trend
- Drill-down Pie Charts: Product-level breakdown
- Interactive Slicers: Product Name, Category, City

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers for interactivity
- Dashboard Design & Formatting

## Insights
- Electronics had the highest order count (~2000+ orders)
- Cash was the most used payment method (₹1,93,000+)
- London showed the highest city-wise revenue
